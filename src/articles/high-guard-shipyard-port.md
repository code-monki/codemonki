This application is a port of Andrea Vallance's (Nicole Susans') High Guard Shipyard 2.0 application. In addition to preserving all of the original functionality the UI has been enhanced with the following features:

- Vue-based responsive web page
- User-selected themes (light, dark, sepia)
- JSON import/export capability
- WCAG 2.2 AA compliance
 
Nicole Susans graciously gave permission for me to create and host the port.
